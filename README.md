# Projekt główny CV

Numer indeksu: 77442

To jest główne repozytorium projektu. Zawiera aktualną wersję strony po zadaniach 4–8.

Funkcje:
- zmiana motywu green/red
- ukrywanie i pokazywanie sekcji
- formularz z walidacją
- pobieranie danych z data.json przez fetch()
- localStorage
- backend Node.js i POST

Backend lokalnie:
```bash
node server.js
```
